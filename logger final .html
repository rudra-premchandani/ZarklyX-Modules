What is Logger

Complete visibility into
every operation and action

ZarklyX Logger is a dual-layer monitoring system that tracks both internal software operations and individual user behaviour — giving you a complete, unbroken record of everything that happens inside your platform.

Two Layers of Complete Monitoring

The ZarklyX Logger module operates on two distinct but complementary layers. The System Logger tracks every internal operation performed within the software — every time someone interacts with the data, a log is created. The Activity Logger tracks user behaviour from the moment they log in, monitoring how a person actually uses the interface. Together, these two layers provide a complete, real-time audit of everything happening inside your platform at any given moment.

A Record You Can Always Trust

Every log entry is timestamped, attributed to a specific user, and preserved in full. Whether you need to investigate a data change, review a user's session, verify a deletion, or audit system activity for compliance purposes, the ZarklyX Logger gives you an accurate, tamper-evident record that you can rely on completely.

How the Logger Works

Automatic logging from
every interaction

No manual entries, no configuration required. Every operation and every user action is captured automatically the moment it occurs.

1

User Interacts with Data

The moment any user creates, updates, or deletes information within the platform, the System Logger captures the operation instantly.

System Logger

Every data operation,
permanently recorded

The System Logger tracks internal operations performed within the software. Every time someone interacts with the data, a log is created.

What the System Logger Records

The System Logger tracks every internal operation — Create, Update, and Delete. Each log entry captures exactly what data was affected, which user performed the action, and the precise timestamp at which the operation occurred.

Why System Logging Matters

System logs are the foundation of data accountability. They answer the fundamental questions: what changed, who changed it, and when. Whether used for compliance auditing, debugging, or dispute resolution, the System Logger ensures no operation goes unrecorded.

Activity Logger

Every user session,
fully visible

The Activity Logger tracks user behaviour from the moment they log in, monitoring how a person actually uses the interface.

What the Activity Logger Tracks

The Activity Logger monitors Login and Logout times, Mouse Movements, Key Presses, and General Activity — providing an overall record of what the user is doing while they are active on the platform.

Session-Level Transparency

Activity logging provides session-level transparency that goes beyond simple login records. By capturing mouse movements, key presses, and general activity patterns, the Activity Logger gives administrators a complete picture of how the platform is being used.

Why ZarklyX Logger

Total transparency,
zero blind spots

ZarklyX Logger is built for platforms that need complete operational visibility and user accountability at every level.

Automatic and Always On

The Logger module runs automatically from the moment the platform is active. There is no manual configuration, no opt-in required, and no gaps in coverage. Every operation is logged the moment it occurs.

Complete Audit Trail for Compliance and Security

Both loggers contribute to a comprehensive audit trail that satisfies the requirements of compliance reviews, internal investigations, and security audits. Every entry is timestamped, attributed to a named user, and stored in a permanent, searchable record.
