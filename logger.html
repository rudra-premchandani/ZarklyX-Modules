What is Logger

Complete visibility into
every operation and action

ZarklyX Logger is a dual-layer monitoring system that tracks both internal software operations and individual user behaviour — giving you a complete, unbroken record of everything that happens inside your platform.

Two Layers of Complete Monitoring

The ZarklyX Logger module operates on two distinct but complementary layers. The System Logger tracks every internal operation performed within the software — every time someone interacts with the data, a log is created. The Activity Logger tracks user behaviour from the moment they log in, monitoring how a person actually uses the interface. Together, these two layers provide a complete, real-time audit of everything happening inside your platform at any given moment.

A Record You Can Always Trust

Every log entry is timestamped, attributed to a specific user, and preserved in full. Whether you need to investigate a data change, review a user's session, verify a deletion, or audit system activity for compliance purposes, the ZarklyX Logger gives you an accurate, tamper-evident record that you can rely on completely.

How the Logger Works

Automatic logging from
every interaction

No manual entries, no configuration required. Every operation and every user action is captured automatically the moment it occurs.

1

User Interacts with Data

System Logger

Every data operation,
permanently recorded

The System Logger tracks internal operations performed within the software. Every time someone interacts with the data, a log is created.

What the System Logger Records

The System Logger tracks every internal operation performed within the software. Every time someone interacts with the data, a log is created automatically. When new information is added to the system, a Create log is generated. When existing information is changed in any way, an Update log is recorded. When information is removed from the system, a Delete log is written. Each of these log entries captures exactly what data was affected, which user performed the action, and the precise timestamp at which the operation occurred.

Why System Logging Matters

System logs are the foundation of data accountability. They answer the fundamental questions any platform must be able to answer: what changed, who changed it, and when. Whether used for internal review, compliance auditing, debugging, or dispute resolution, the System Logger ensures that no operation goes unrecorded and no change is ever untraceable.

Activity Logger

Every user session,
fully visible

The Activity Logger tracks user behaviour from the moment they log in, monitoring how a person actually uses the interface.

What the Activity Logger Tracks

The Activity Logger monitors how a person actually uses the interface throughout their session. It records Login and Logout times, capturing exactly when a user starts and ends their session. It tracks Mouse Movements, recording where the user moves their cursor on the screen during their session. It logs Key Presses, capturing any typing or keyboard activity that takes place. It also monitors General Activity, providing an overall record of what the user is doing while they are active on the platform.

Session-Level Transparency

Activity logging provides session-level transparency that goes beyond simple login records. By capturing mouse movements, key presses, and general activity patterns, the Activity Logger gives administrators a complete picture of how the platform is being used — enabling productivity analysis, security monitoring, and behavioural auditing without requiring any action from the user being monitored.
